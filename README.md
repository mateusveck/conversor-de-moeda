# 💱 Conversor de Moedas

Este é o meu primeiro projeto utilizando **JavaScript** integrado com HTML e CSS.

O objetivo do projeto é criar um conversor simples de moedas, onde o usuário pode inserir um valor, selecionar a moeda desejada e obter a conversão para reais (BRL).

---

## 🚀 Funcionalidades

- Seleção de diferentes moedas (Dólar, Euro, Libra, Iene, etc.)
- Conversão automática para Real (BRL)
- Formatação de valores no padrão brasileiro
- Validação simples de input
- Exibição dinâmica do resultado
- Interface responsiva e estilizada

---

## 🧠 O que aprendi com este projeto

Durante o desenvolvimento, pude praticar e entender melhor:

- Manipulação do DOM com JavaScript
- Eventos (`input`, `change`, `submit`)
- Condicionais (`switch`, `if`)
- Conversão e formatação de números
- Organização de lógica no front-end
- Interação entre HTML, CSS e JS
